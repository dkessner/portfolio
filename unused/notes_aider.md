# notes aider

## Configuration

`.aider.conf.yml` 

Config files loaded from locations in this order, last config loaded takes
priority:
- home directory
- git repo root
- current directory


```
model: ollama_chat/deepseek-coder-v2
auto-commits: false

read:
  - specifications.md
```

[Reference](https://aider.chat/docs/config/aider_conf.html)
